🚀 Dual AI Vision Platform
Real-Time Sign Language + Object Detection System

An integrated AI-powered web platform that combines Sign Language Recognition and YOLO-based Object Detection into a unified real-time experience.

This project demonstrates practical implementation of Computer Vision, Deep Learning, and Interactive AI Systems inside a browser-based environment.

🧠 Overview

The Dual AI Vision Platform bridges accessibility and visual intelligence by merging gesture detection and object recognition into one scalable architecture.

It enables users to:

Practice and detect sign language in real time

Detect surrounding objects via webcam or media upload

View confidence scores and detection logs

Track performance through an analytics dashboard

🔥 Key Features

✅ Real-Time Sign Language Detection (CNN + MediaPipe)

✅ YOLO-Based Object Detection

✅ Live Webcam Streaming

✅ Confidence Score Visualization

✅ Sentence Builder (Sign AI)

✅ User Dashboard & Analytics

✅ Time Tracking System

✅ Detection History Logs

✅ Leaderboard (Top Users)

✅ Fully Browser-Based (No Installation Required)

🧩 System Architecture

The platform follows a modular AI architecture:

Sign AI Module → Hand landmark detection → Gesture classification → Confidence scoring

Object AI Module → YOLO inference → Bounding box rendering → Label prediction

User Analytics Layer → Session tracking → Performance logging → Ranking logic

This modular structure allows future expansion into multimodal contextual AI systems.
